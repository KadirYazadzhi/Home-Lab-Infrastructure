# Service: Uptime Kuma

### 🧐 What is it?
A self-hosted monitoring tool that checks the status of your services in real-time.

### 💡 Why is it useful?
It provides "Peace of Mind." You get notified immediately via Telegram or Discord if your Storage Node (Node C) or any critical service like OpenVPN goes offline.

### ⚖️ Pros & Cons
*   **Pros:** Beautiful status pages, supports 50+ notification methods.
*   **Cons:** Only tells you if a service is "up", not necessarily if it's "healthy" internally.
