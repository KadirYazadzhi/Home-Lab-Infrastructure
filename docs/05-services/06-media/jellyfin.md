# Service: Jellyfin

### 🧐 What is it?
A media server that organizes and streams movies, shows, and music.

### 💡 Why is it useful?
It utilizes the **1TB HDD on Node C**. It allows you to stream your local movie library to any device in the house.

### ⚖️ Pros & Cons
*   **Pros:** 100% open source, supports hardware transcoding (Intel QuickSync).
*   **Cons:** TV apps can be buggy compared to Plex.
