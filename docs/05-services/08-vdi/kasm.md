# Service: Kasm Workspaces

### 🧐 What is it?
A platform for containerized desktop infrastructure (VDI). It streams "Desktops" or "Browsers" to your browser.

### 💡 Why is it useful?
- **Disposable Browsing:** Open a bank site in a fresh container, then destroy it.
- **Remote Desktop:** Run a full Linux desktop from an iPad.

### 🏗 Components
Includes `kasm_proxy`, `kasm_manager`, `kasm_agent`, `kasm_api`, `kasm_share`, `kasm_guac`, `kasm_db`, `kasm_redis`.

### ⚖️ Pros & Cons
*   **Pros:** Unbeatable isolation and security.
*   **Cons:** Massively complex setup, requires significant RAM.
